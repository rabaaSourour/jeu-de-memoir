🧠🌊 Jeu de Mémoire - Monde Aquatique
📖 Description

Ce projet est un jeu de memory interactif et immersif inspiré du monde marin.
Il combine gameplay classique et expérience visuelle avancée grâce à des animations fluides, un environnement dynamique et des effets interactifs.

Le joueur doit retrouver toutes les paires de cartes tout en évoluant dans un univers aquatique animé 🐠

✨ Fonctionnalités principales :
🎴 Système de cartes à retourner (memory classique)
🔀 Mélange aléatoire des cartes
🎯 Détection automatique des paires
🏆 Écran de victoire avec animation + confettis
🔁 Bouton de redémarrage dynamique
🎨 Interface immersive (effets marins, lumière, animations)
🖱️ Curseur personnalisé animé
🌊 Effets visuels (flottement, blur, lumière sous-marine)
🐟 Animation 3D avec Three.js (poisson interactif)

🧠 Gameplay :
Clique sur deux cartes pour les retourner
Si elles sont identiques → elles restent visibles
Sinon → elles se retournent
Trouve toutes les paires pour gagner
